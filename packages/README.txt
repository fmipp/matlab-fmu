In addition to subdirectoy "+fmipputil", this directory should contain the following SWIG generated content:
- directory: +fmippex
- directory: +fmippim
- file: SwigMem.m
- file: SwigRef.m

In addition, subdirectory "lib" should contain the following libraries (all compiled in Release mode):
- fmippex.dll
- fmippexMEX.mexw32
- fmippim.dll
- fmippimMEX.mexw32
